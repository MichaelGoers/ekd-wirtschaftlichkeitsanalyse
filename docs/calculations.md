# EKD Wirtschaftlichkeitsanalyse

## Berechnungsregeln

### 1. Aktuelle Situation

Jährliche Kosten

```
Stromkosten
+ Gaskosten
= Aktuelle jährliche Kosten
```

20 Jahre

```
Aktuelle jährliche Kosten
× Betrachtungszeitraum
```

---

### 2. Investition

```
Wärmepumpenlösung
- KfW-Förderung
+ Photovoltaiklösung
```

---

### 3. Wärmepumpe

```
Hausverbrauch

+

Wärmepumpenverbrauch

=

Gesamtverbrauch

×

Strompreis
```

---

### 4. Wärmepumpe + PV

Einspeisung

```
PV-Ertrag
× Einspeiseanteil
```

Einspeisevergütung

```
Einspeisung
× Einspeisevergütung
```

Netzbezug

```
(Hausverbrauch + Wärmepumpenverbrauch)
× Netzbezugsanteil
```

Netzkosten

```
Netzbezug
× Strompreis
```

Jährliche Kosten

```
Netzkosten
- Einspeisevergütung
```

---

### 5. Wärmepumpe + PV + EKDFlow

Wie Wärmepumpe + PV

zusätzlich

```
Netzkosten
- Reduzierung Netzentgelte
```

---

### 6. Ersparnis

```
Aktuelle Kosten über Betrachtungszeitraum

-

Investition

-

Kosten Wärmepumpe/PV/EKDFlow
```
